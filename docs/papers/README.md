# Papers and Research Outputs

Use this directory for internal reports, paper drafts, rebuttal notes, and
sanitized release plans.

Recommended layout:

```text
docs/papers/
  scaling-laws-for-moe-recsys/
    outline.md
    draft.md
    figures.md
    reviews.md
```

Keep raw experiment records in `experiments/` and large artifacts outside Git.
Paper drafts should link to experiment IDs and artifact manifests rather than
copying raw logs.
